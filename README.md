# Detections.ai Project

A personal detection engineering repository used to store, organize, and publish detection rules created or improved through detections.ai.

The goal of this repository is to maintain practical, readable, and community-ready detections with a focus on behavioral detection logic, Microsoft Defender XDR, KQL, Sigma, and SOC investigation use cases.

---

## Purpose

This repository is used for:

- Storing detections created with the help of detections.ai
- Organizing detection rules by language and platform
- Maintaining public detection engineering work for community contribution
- Building a personal portfolio of practical detection logic
- Improving detections through review, tuning, and validation
